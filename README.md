# namegen.py

generates names for my various projects using a python script. I might build it into a js script to be used for my website at some point.

## usage

```bash
# 1 random codename
$ ./namegen.py
# 5 random codenames
$ ./namegen.py 5
# 1 ULTRA-prefixed codename
$ ./namegen.py 1 ULTRA
# 5 ULTRA-prefixed codenames
$ ./namegen.py 5 ULTRA
```
